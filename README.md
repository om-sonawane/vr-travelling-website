# 🌍 Virtual Reality Traveling Website

Welcome to the **Virtual Reality Traveling Website**! This project brings immersive travel experiences to users through stunning 360° panoramic images. Explore beautiful locations around the world from the comfort of your home using **VR technology**. We've integrated **Momento360** to provide a seamless viewing experience of 360° content.

## 🏞️ Key Features

- **Panoramic 360° Images**: Travel to breathtaking destinations with full 360° views.
- **VR Compatibility**: Experience the website in Virtual Reality for more immersion (works with popular VR headsets like Oculus).
- **Smooth Embedding**: Seamlessly embedded 360° images via **Momento360** for enhanced user experience.
- **Responsive Design**: Optimized for both desktop and mobile, ensuring a smooth and immersive experience on any device.
  
## 🎯 Project Overview

The Virtual Reality Traveling Website allows users to virtually visit a selection of tourist destinations, thanks to panoramic 360° images. The core functionality is based on:

1. **360° Panoramic View**: Full immersive experience through embedded images.
2. **Easy Navigation**: Users can explore different locations through an interactive interface.
3. **VR Support**: The website supports VR mode, enabling users to wear a headset for even deeper immersion.

## 🛠️ Technologies Used

- **HTML5** for the website structure 🌍
- **CSS3** for the styling and responsive design 🎨
- **JavaScript** for interactive elements and smooth transitions 🧩
- **Momento360** for embedding 360° panoramic images seamlessly into the website 🖼️

## 🌐 Website Structure

Home Page: An introduction to virtual reality traveling, showcasing featured 360° travel destinations.
Destination Pages: Each page focuses on a unique destination, offering users the ability to view and interact with 360° images.
VR Mode: The website is optimized for VR, so users with headsets can dive into the immersive experience.

## 🎮 How to Use

Browse Destinations: Navigate through the various available travel destinations.
Interact with 360° Images: Click and drag on the 360° images to explore every angle of the scene.
Activate VR Mode: For users with VR headsets, click the VR Mode button to switch into immersive mode.

## 🔗 Embedding with Momento360

We have used Momento360 to embed panoramic 360° images into the website. Here's an example of how embedding is done:
1. **Embedding process**:
   ```bash
   <iframe width="100%" height="500px" src="https://momento360.com/e/u/example-link/?utm_campaign=embed&utm_source=other" frameborder="0" allowfullscreen></iframe>
This allows users to explore the panoramic views with ease directly on the webpage.
